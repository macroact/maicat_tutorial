# Maicat Tutorial
# Maicat Gazebo

# Run the Gazebo environment: 
```python
    ros2 launch maibot_gazebo maibot_gazebo.launch.py
```

# Run nav2:
```python
    ros2 launch maibot_navigation2 navigation.launch.py
```
To navigate:
- Click '2D Nav Goal'.
- Click and drag at the position you want the robot to go.
